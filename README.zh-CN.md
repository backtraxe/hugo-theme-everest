# Everest | Hugo 主题

[English](https://github.com/backtraxe/hugo-theme-everest/blob/master/README.md) | 中文

## 简介

Everest 是 Hugo 的博客主题。它几乎包含了博客主题应该具备的所有功能，包括搜索、画廊等等。它看起来简单，但一旦你深入了解，就会发现它有许多功能。

<br>

## 特征

- 多主题
- 移动菜单
- 搜索
- 搜索引擎优化（SEO）
- 多语言支持（国际化）
- 响应式设计
- 可定制的用户界面
- RSS
- 画廊
- 快速代码高亮
- 演讲页面
- 展示页面
- 发布页面
- 简历页面
- 演示页面
